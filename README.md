## Clone The Repository
* **Clone with HTML**

```
git clone https://gitlab.emi.u-bordeaux.fr/pt2/teams/tm01/takuzu-01c.git
```

* **Clone with SSH**

```
git clone git@gitlab.emi.u-bordeaux.fr:pt2/teams/tm01/takuzu-01c.git
```

## Usage
* **Build the project**

```
mkdir build; cd build; cmake ..; make
```

* **Run the game**

```
./game_text
```
